# Patch Review Board Job Instructions

This repository contains the job instructions for the AI Agent acting as a member of the Patch Review Board.
The instructions detail the process for quarterly OS patch recommendation for Linux (RHEL, Ubuntu, Oracle Linux).

## Contents
- **PRB_Instruction-Linux.md**: Detailed job instruction document.

## Usage
These instructions are intended to be used by an AI Agent (e.g., OpenClaw) to perform patch research and analysis.
